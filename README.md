# 📊 Sales Data Analysis using Python

## 📌 Project Overview
This project analyzes retail sales data to uncover business insights related to product categories, regional performance, and profitability.

## 🎯 Objectives
- Identify the most profitable product category
- Analyze regional sales performance
- Understand profit distribution across categories
- Explore sub-category level insights

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset
Sample Superstore dataset containing:
- Category & Sub-Category
- Sales & Profit
- Region & Customer Segment

## 📊 Analysis Performed
- Data cleaning and preprocessing
- Grouping and aggregation using Pandas
- Visualization using Matplotlib & Seaborn

## 📈 Key Insights
- Technology category generated the highest profit
- West region recorded the highest sales
- Some categories show high sales but lower profit, indicating cost or discount impact
- Sub-category analysis highlights top-performing products
